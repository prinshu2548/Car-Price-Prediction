# Car-Price-Prediction
End-to-end Car Price Prediction system using Python, Pandas, NumPy, Scikit-Learn, and data visualization techniques.

## Overview
This project predicts the selling price of used cars using Machine Learning.

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Model Used
- Random Forest Regressor

## Dataset Features
- Car Name
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner
